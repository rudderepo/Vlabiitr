ADC Simulators - Experimental Procedure

Experimental Procedure

This experiment has three different Analog-to-Digital Converters (ADCs). Follow these steps for each ADC type.

Single Slope ADC Procedure

  1. Set ADC Parameters:
    • Select number of bits for output
    • Set max count time using slider (10ms to 100ms)
    • Clock frequency will be calculated automatically
    • Select op-amp supply voltage
    • Set reference voltage (Vref)
    • Choose capacitor value
    • Adjust input voltage (Vin)
  2. Calculate RC Parameters: Click "Calculate RC Parameters" button
  3. Make Circuit Connections: Connect all 11 circuit points. Use "Auto-Connect" if needed.
  4. Check Connections: Click "Check Connections" to verify
  5. Run Simulation: Click "Run Simulation" to see:
    • Ramp voltage increasing
    • Counter counting
    • Digital output when ramp crosses input

SAR ADC Procedure

  1. Set ADC Parameters:
    • Select number of bits
    • Set reference voltage (Vref)
    • Adjust input voltage (Vin)
    • Choose step speed
  2. Make Circuit Connections: Connect 7 circuit points. Use "Auto-Connect" if needed.
  3. Check Connections: Click "Check Connections"
  4. Start Simulation: Click "Start Simulation"
  5. Step-by-Step Conversion: Use buttons:
    • Next Step: Move to next bit
    • Previous Step: Go back
    • Auto Run: Automatic steps
    • Reset: Start over

Dual Slope ADC Procedure

  1. Set ADC Parameters:
    • Select number of bits
    • Set reference voltage (Vref)
    • Adjust input voltage (Vin)

    Note: T1 integration time is fixed at 20ms

  2. Make Circuit Connections: Connect 12 circuit points. Use "Auto-Connect" if needed.
  3. Check Connections: Verify all connections
  4. Run Simulation: Watch two phases:
    • Phase 1: Integrate with Vin for 20ms
    • Phase 2: Deintegrate with -Vref
    • Counter counts only in Phase 2

General Instructions

Things to Observe:

  • Compare conversion speeds
  • Note circuit complexity
  • Check accuracy of each ADC

Tips:

  • Try different input voltages
  • Change bit resolution
  • Compare all three ADCs
  • Use "Reset All" for new test
  • Complete Post-Test after simulation
