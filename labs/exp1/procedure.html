ADC Simulators - Experimental Procedure

Experimental Procedure

This experiment includes three different Analog-to-Digital Converters (ADCs). Follow these systematic steps for each ADC type to understand their working principles and characteristics.

1. Single Slope ADC Procedure

Step 1: Set ADC Parameters
  • Select number of bits for digital output
    • Figure 1: Bit resolution selection for Single Slope ADC
  • Set maximum count time using slider (10ms to 100ms)
    • Figure 2: Max count time adjustment
  • Clock frequency is calculated automatically based on bits and timing
  • Select op-amp supply voltage (±V)
    • Figure 3: Op-amp supply voltage setting
  • Set reference voltage (Vref)
    • Figure 4: Reference voltage setting
  • Choose integrator capacitor value (µF)
    • Figure 5: Capacitor value selection
  • Adjust input voltage (Vin) using slider
    • Figure 6: Input voltage adjustment
Step 2: Calculate RC Parameters

Click the "Calculate RC Parameters" button to compute required resistance and ramp slope.

Figure 7: RC parameters calculation button
Step 3: Make Circuit Connections

Connect all 11 circuit points as indicated by the red dots. Use "Auto-Connect" button for automatic wiring.

Step 4: Check Connections

Click "Check Connections" to verify all 11 connections are correct.

Figure 8: Complete Single Slope ADC circuit
Step 5: Run Simulation

Click "Run Simulation" to observe:

  • Ramp voltage increasing linearly
  • Counter incrementing
  • Digital output when ramp crosses input voltage
Figure 9: Run simulation button
